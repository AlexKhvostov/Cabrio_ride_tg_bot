# 🚗 Cabrio Club Telegram Bot

Telegram бот для управления клубом любителей кабриолетов. Полнофункциональная система управления участниками, автомобилями и приглашениями.

## 🌟 Основные возможности

### 👥 Управление участниками
- **Регистрация** с фотографией профиля
- **Автоматическое отслеживание** выхода/возвращения участников
- **Умная статистика** (исключает вышедших участников)
- **Эмоциональные уведомления** о выходе и возвращении

### 🚗 Управление автомобилями
- **Добавление автомобилей** с множественными фотографиями
- **Просмотр автомобилей** с фото и подробной информацией
- **Поиск по номерам** (полный и частичный поиск)
- **Статусы автомобилей** (активный, продан, в ремонте, разбит)

### 🎯 Система приглашений
- **Создание приглашений** для незнакомых кабриолетов
- **Фотографии приглашений** с геолокацией
- **Отслеживание статуса** приглашений
- **Групповые уведомления** с фото

### 📊 Статистика и аналитика
- **Статистика клуба** (участники, автомобили, приглашения)
- **Исключение вышедших** из основной статистики
- **Детальная информация** по каждому участнику

## 🛠 Технологии

- **Node.js** - основная платформа
- **node-telegram-bot-api** - работа с Telegram API
- **MySQL** - база данных
- **dotenv** - управление переменными окружения

## 📦 Установка

### 1. Клонирование репозитория
```bash
git clone https://github.com/AlexKhvostov/Cabrio_ride_tg_bot.git
cd Cabrio_ride_tg_bot
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корне проекта:
```env
# Telegram Bot
BOT_TOKEN=your_bot_token_here
CHAT_ID=your_chat_id_here

# Database
DB_HOST=your_database_host
DB_USER=your_database_user
DB_PASSWORD=your_database_password
DB_NAME=your_database_name
DB_PORT=3306
```

### 4. Настройка базы данных
Выполните SQL скрипты из папки `sql/`:
```bash
# Основная структура
mysql -u username -p database_name < sql/database_structure.sql

# Миграция для поля left_date
mysql -u username -p database_name < sql/add_left_date_migration.sql
```

### 5. Запуск бота
```bash
node src/bot/bot.js
```

## 🔧 Настройка Telegram бота

### 1. Создание бота
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Используйте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Получите токен бота

### 2. Настройка группы
1. Создайте группу для клуба
2. Добавьте бота в группу
3. Сделайте бота администратором (для отправки сообщений)
4. Получите CHAT_ID группы:
   - Напишите любое сообщение в группе
   - Перейдите: `https://api.telegram.org/bot<BOT_TOKEN>/getUpdates`
   - Найдите `"chat":{"id":-1001234567890}` и скопируйте ID

## 📸 Работа с фотографиями

Бот поддерживает загрузку и отображение фотографий:
- **Фото профиля** - при регистрации
- **Фото автомобилей** - множественные фото для каждого авто
- **Фото приглашений** - фото незнакомых кабриолетов
- **Автоматическое создание директорий** для хранения
- **Fallback на текст** при отсутствии фото

## 🔄 Система участников

### Автоматическое отслеживание
- **Выход из группы** - автоматически ставит статус "вышел"
- **Возвращение в группу** - восстанавливает статус "активный"
- **Умные уведомления** - различает новых и вернувшихся участников

### Статусы участников
- `новый` - только зарегистрировался
- `активный` - активный участник клуба
- `вышел` - покинул группу (данные сохраняются)

## 📊 База данных

### Основные таблицы
- `members` - участники клуба
- `cars` - автомобили участников
- `invitations` - приглашения для незнакомых авто

### Ключевые поля
- `left_date` - дата выхода из клуба (для отслеживания)
- `photos` - JSON массив фотографий
- `status` - статус участника/автомобиля
