# 📁 Реструктуризация проекта

## Обзор изменений

Проект был реорганизован для улучшения структуры и удобства разработки. Все файлы были перемещены в логические каталоги.

## 🔄 Изменения в структуре

### До реструктуризации
```
📦 tg_bot/
├── bot.js                    # Основной файл бота
├── database.js               # База данных
├── config.js                 # Конфигурация
├── config-debug.js           # Отладочная конфигурация
├── *.md                      # Документация
├── Procfile                  # Деплой файлы
├── railway.toml
├── app.js
├── server.js
└── uploads/                  # Загруженные файлы
```

### После реструктуризации

## 🔧 Технические изменения

### 1. Новая точка входа
- **Создан**: `index.js` - главный файл для запуска приложения
- **Обновлен**: `package.json` - изменен main файл с `bot.js` на `index.js`
- **Обновлены**: скрипты npm для использования нового файла

### 2. Исправлены импорты
- **bot.js**: обновлены пути импорта config и database
- **database.js**: обновлен путь импорта config
- **config.js**: обновлены пути к папке uploads

### 3. Обновлены файлы деплоя
- **Procfile**: изменен с `bot.js` на `index.js`
- **app.js**: обновлен путь импорта
- **server.js**: обновлен путь импорта

### 4. Организация документации
Вся документация перемещена в папку `docs/`:
- Руководства пользователя
- Техническая документация
- История изменений
- Результаты аудита

### 5. Структурирование SQL
Создана папка `sql/` для:
- SQL скриптов создания таблиц
- Диаграмм базы данных
- Примеров данных

## 📋 Преимущества новой структуры

1. **Логическая организация**: код разделен по функциональности
2. **Масштабируемость**: легко добавлять новые модули
3. **Читаемость**: понятная структура для новых разработчиков
