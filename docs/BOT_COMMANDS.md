# 🤖 Команды бота клуба кабриолетов

Полный список команд Telegram бота с описанием и примерами использования.

## 🚀 Основные команды

### `/start`
**Описание:** Запуск бота и приветствие  
**Использование:** Первая команда для новых пользователей  
**Результат:** 
- Проверка регистрации пользователя
- Приветственное сообщение
- Предложение регистрации для новых пользователей

### `/menu`
**Описание:** Главное меню с интерактивными кнопками  
**Использование:** Быстрый доступ ко всем функциям  
**Результат:** Клавиатура с кнопками для всех основных функций

### `/register`
**Описание:** Регистрация в клубе  
**Использование:** Для новых участников  
**Процесс:** Имя → Фамилия → Город → Телефон → О себе → Фото  
**Результат:** Создание профиля участника + уведомление в группу

### `/help`
**Описание:** Справка по всем командам  
**Использование:** Когда нужна помощь  
**Результат:** Полный список команд с описанием

## 👤 Профиль и автомобили

### `/profile`
**Описание:** Просмотр своего профиля  
**Показывает:** Имя, статус, дату вступления, контакты, описание  
**Дополнительно:** Кнопки для редактирования

### `/cars`
**Описание:** Список своих автомобилей  
**Показывает:** Марка, модель, год, номер, статус, описание  
**Если нет авто:** Предложение добавить первый автомобиль

### `/addcar`
**Описание:** Добавление нового автомобиля  
**Процесс:** Марка → Модель → Год → Номер → Фото  
**Результат:** Добавление авто + уведомление в группу

## 🎯 Система приглашений

### `/invite`
**Описание:** Создание приглашения (фиксация визитки)  
**Процесс:** Номер → Фото → Место → Марка → Модель → Контакты → Заметки  
**Результат:** Сохранение приглашения + уведомление в группу

### `/myinvites`
**Описание:** Просмотр своих приглашений  
**Показывает:** Список всех приглашений со статистикой  
**Группировка:** По статусам (новое, контакт, встреча и т.д.)

### `/search`
**Описание:** Поиск автомобилей по номеру  
**Возможности:** Полный или частичный поиск  
**Примеры:** `A123BC77`, `A123`, `BC77`, `77`  
**Результат:** Список найденных автомобилей с деталями

## 📊 Информация

### `/stats`
**Описание:** Статистика клуба  
**Показывает:** 
- Количество участников (всего/активных)
- Количество автомобилей
- Статистика приглашений
- Конверсия приглашений

## 🔧 Служебные команды

### `/skip`
**Описание:** Пропустить текущий шаг в процессе  
**Использование:** Во время регистрации, добавления авто, создания приглашения  
**Контекст:** Работает только в активных процессах

### `/done`
**Описание:** Завершить загрузку фотографий  
**Использование:** После загрузки одного или нескольких фото  
**Контекст:** Этап загрузки фото в любом процессе

## 📱 Автодополнение команд

Бот автоматически регистрирует команды в Telegram, поэтому:
- ✅ Команды появляются в автодополнении при вводе `/`
- ✅ Каждая команда имеет описание с эмодзи
- ✅ Удобный выбор из списка

## 🔄 Состояния и процессы

Некоторые команды запускают многошаговые процессы:

### Регистрация (`/register`)
```
Состояние: registering
Этапы: name → last_name → city → phone → about → photo
Команды: /skip (на любом этапе)
```

### Добавление автомобиля (`/addcar`)
```
Состояние: adding_car
Этапы: brand → model → year → reg_number → photos
Команды: /skip (кроме обязательных), /done (на этапе фото)
```

### Создание приглашения (`/invite`)
```
Состояние: creating_invitation
Этапы: reg_number → photos → location → brand → model → contact_info → notes
Команды: /skip, /done (фото), /finish (с 3-го этапа)
```

### Поиск (`/search`)
```
Состояние: searching
Этапы: number (ввод номера для поиска)
Команды: любой текст как поисковый запрос
```

## 🛡️ Безопасность команд

- ✅ Все команды требуют членства в группе клуба
- ✅ Проверка доступа перед выполнением
- ✅ Обработка ошибок и некорректного ввода
- ✅ Автоматическая очистка состояний при ошибках

## 💡 Советы по использованию

1. **Новичкам:** Начните с `/start` и `/menu`
2. **Быстрый доступ:** Используйте прямые команды `/profile`, `/cars` и т.д.
3. **В процессах:** Помните про `/skip` и `/done`
4. **При ошибках:** Используйте `/help` или `/menu`
5. **Поиск:** `/search` поддерживает частичный поиск
