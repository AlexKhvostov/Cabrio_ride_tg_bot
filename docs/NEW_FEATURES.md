# Новые функции бота клуба кабриолетов

## 1. Автоматическое изменение статуса при выходе из группы

### Описание
Когда участник покидает группу Telegram, бот автоматически:
- Изменяет статус участника на "вышел"
- Добавляет дату выхода (`left_date`)
- Изменяет статус всех автомобилей участника на "вышел"
- Отправляет уведомление в группу

### Технические детали
- **Событие:** `left_chat_member`
- **Проверки:** ID группы, исключение ботов
- **Обновления БД:** статус участника и всех его автомобилей
- **Уведомления:** сообщение в группу с количеством автомобилей

### Влияние на статистику
- **Участники в клубе:** исключает вышедших (`status != "вышел"`)
- **Автомобили в клубе:** исключает авто вышедших (`status != "вышел"`)
- **Новые поля:** показывает количество вышедших участников и их авто

### Поиск
- Вышедшие участники и их автомобили остаются доступными в поиске
- Это позволяет не класть визитки повторно при возвращении

## 2. Отображение фотографий в профиле и автомобилях

### Профиль пользователя
- **Команда:** `/profile` или кнопка "👤 Мой профиль"
- **Функция:** `showUserProfile()`
- **Логика:**
  - Если есть `photo_url` - отправляет фото с подписью
  - Если файл не найден - текст + уведомление об ошибке
  - Если фото нет - только текст
- **Путь к фото:** `uploads/members/[filename]`

### Автомобили пользователя
- **Команда:** `/cars` или кнопка "🚗 Мои авто"
- **Функция:** `showUserCars()`
- **Логика:**
  - Каждый автомобиль отправляется отдельным сообщением
  - Если есть фото - показывает первое фото из списка
  - Указывает количество фото (например, "Фото: 1 из 3")
  - Задержка 500мс между сообщениями
- **Путь к фото:** `uploads/cars/[filename]`
- **Формат хранения:** список файлов через запятую в поле `photos`

### Обработка ошибок
- Проверка существования файлов
- Обработка ошибок чтения/отправки
- Информативные сообщения об ошибках
- Fallback на текстовые сообщения

## 3. Изменения в базе данных

### Новое поле в таблице members
```sql
ALTER TABLE members 
ADD COLUMN left_date DATE NULL COMMENT 'Дата выхода из клуба' 
AFTER join_date;
```

### Обновление статуса автомобилей
```sql
ALTER TABLE cars 
MODIFY COLUMN status ENUM('активный', 'продан', 'в ремонте', 'разбит', 'приглашение', 'вышел') DEFAULT 'активный';
```

### Новый индекс
```sql
CREATE INDEX idx_members_left_date ON members(left_date);
```

## 4. Обновленная статистика

### Новые показатели
- **Участников в клубе:** только активные (исключая вышедших)
- **Автомобилей в клубе:** только активные авто
- **Вышло из клуба:** количество вышедших участников
- **Авто вышедших:** количество автомобилей вышедших участников

### SQL запросы
```sql
-- Активные участники
SELECT COUNT(*) FROM members WHERE status != "вышел"

-- Активные автомобили  
SELECT COUNT(*) FROM cars WHERE status != "вышел"

-- Вышедшие участники
SELECT COUNT(*) FROM members WHERE status = "вышел"

-- Автомобили вышедших
SELECT COUNT(*) FROM cars WHERE status = "вышел"
```

## 6. Тестирование

### Тест выхода из группы
1. Добавить тестового пользователя в группу
2. Зарегистрировать его через бота
3. Добавить автомобиль
4. Удалить из группы
5. Проверить статусы в БД
6. Проверить статистику

### Тест отображения фотографий
1. Загрузить фото профиля при регистрации
2. Загрузить фото автомобиля
3. Проверить команду `/profile`
4. Проверить команду `/cars`
5. Проверить обработку отсутствующих файлов

## 7. Совместимость

### Обратная совместимость
- Старые записи без `left_date` работают корректно (NULL значения)
- Автомобили без статуса "вышел" не затронуты
- Профили без фото отображаются как раньше

### Требования к серверу
- Доступ к файловой системе для чтения фото
- Права на чтение папок `uploads/members/` и `uploads/cars/`
- Поддержка отправки фото через Telegram Bot API
