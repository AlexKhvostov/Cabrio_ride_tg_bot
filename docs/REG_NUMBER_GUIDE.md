# 🔢 Руководство по регистрационным номерам

## Новая функция: Регистрационные номера автомобилей

### 📋 Что добавлено:

**В процесс добавления автомобиля:**
1. Марка → Модель → Год
2. **🔢 Регистрационный номер** (новый шаг!)
3. Фотографии → Сохранение

### ✅ Правила ввода номера:

**Разрешенные символы:**
- Латинские буквы: A-Z
- Цифры: 0-9
- Без пробелов, тире, точек

**Длина номера:**
- Минимум: 4 символа
- Максимум: 12 символов

**Примеры правильных номеров:**
```
✅ A123BC77
✅ 1234AB199  
✅ H001AA
✅ ABC1234
✅ 777AAA
```

**Примеры неправильных номеров:**
```
❌ A-123-BC-77  (содержит тире)
❌ А123ВС77     (кириллические буквы)
❌ A123 BC77    (содержит пробел)
❌ A12          (слишком короткий)
❌ A123BC77RUS99 (слишком длинный)
```

### 🔄 Автоматическая обработка:

- **Приведение к верхнему регистру:** `a123bc77` → `A123BC77`
- **Удаление пробелов:** `A123 BC77` → ❌ (ошибка валидации)
- **Проверка формата:** Мгновенная валидация при вводе

### 🎯 Использование:

**При добавлении автомобиля:**
1. Введите номер: `A123BC77`
2. Или пропустите: `/skip`
3. Номер сохраняется в поле `reg_number` в БД

**При просмотре автомобилей:**
```
🚗 Ваши автомобили (1)

1. BMW 3 Series
📅 Год: 2010
🔢 Номер: A123BC77
📊 Статус: активный
```

### 🗄️ База данных:

**Поле:** `cars.reg_number`
- **Тип:** VARCHAR(20)
- **Обязательное:** Нет (NULL разрешен)
- **Формат в БД:** Верхний регистр без пробелов

### 🛠️ Техническая реализация:

**Функция валидации:**
```javascript
function validateRegNumber(regNumber) {
    // Проверка формата, длины, символов
    // Возвращает: { valid: true/false, number/error }
}
```

**Использование в коде:**
```javascript
const validation = validateRegNumber(userInput);
if (validation.valid) {
    data.reg_number = validation.number;
} else {
    // Показать ошибку: validation.error
}
```

### 🎨 Отображение:

**В списке автомобилей:**
- Показывается только если номер указан
- Формат: `🔢 Номер: A123BC77`

**В подтверждении добавления:**
- Включается в сводку нового автомобиля
- Пропускается если номер не указан

### 💡 Преимущества:

1. **Гибкость:** Поддержка разных форматов номеров
2. **Валидация:** Мгновенная проверка корректности
3. **Опциональность:** Можно пропустить если номера нет
4. **Стандартизация:** Автоматическое приведение к единому формату
5. **Безопасность:** Защита от некорректных данных

Теперь пользователи могут указывать регистрационные номера своих автомобилей! 🚗✨
