# Обновление примера номера телефона

## Изменение
Обновлен пример номера телефона в процессе регистрации с российского на белорусский.

## До изменения:
```
Введите ваш номер телефона 📱
Например: +7 (999) 123-45-67
(или нажмите /skip)
```

## После изменения:
```
Введите ваш номер телефона 📱
Например: +375 (33) 993-22-88
(или нажмите /skip)
```

## Обоснование
- Клуб кабриолетов базируется в Минске, Беларусь
- Страна по умолчанию в регистрации: "Беларусь"
- Логично использовать белорусский номер как пример

## Технические детали
- **Файл:** `src/bot/bot.js`
- **Строка:** 990
- **Функция:** `handleRegistration()` в шаге 'phone'

## Формат белорусских номеров
- **Код страны:** +375
- **Мобильные операторы:**
  - МТС: +375 (33) xxx-xx-xx
  - A1: +375 (44) xxx-xx-xx  
  - life:): +375 (25) xxx-xx-xx

## Совместимость
- Изменение не влияет на валидацию (её нет)
- Пользователи могут вводить номера в любом формате
- Обратная совместимость сохранена
