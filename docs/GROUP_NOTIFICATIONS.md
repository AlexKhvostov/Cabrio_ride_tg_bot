# 📢 Уведомления в группу

Бот автоматически отправляет уведомления в группу клуба о важных событиях.

## 🤖 Добавление бота в группу

### ✅ В правильную группу клуба

Когда бота добавляют в группу клуба, он отправляет приветственное сообщение:

```
🤖 Привет всем!

Я бот клуба кабриолетов! 🚗💨

Мои функции:
• 👤 Регистрация участников клуба
• 🚗 Управление автомобилями участников
• 🎯 Система приглашений новых участников
• 📊 Статистика и аналитика клуба
• 🔍 Поиск автомобилей по номерам

Как начать:
Напишите мне в личные сообщения команду /start

Важно: Я работаю только для участников этой группы!
🆔 ID группы: -1001613926754

Добро пожаловать в клуб! 🎉
```

### ❌ В чужую группу (защита)

Если бота добавляют не в ту группу, он отправляет информационное сообщение и покидает группу:

```
🤖 Привет!

Я бот клуба кабриолетов, но работаю только для участников нашей группы.

🚗 Если вы любитель кабриолетов, присоединяйтесь к нам:
👥 https://t.me/Cabrio_Ride

Там вы сможете:
• Общаться с единомышленниками
• Делиться фотографиями своих авто
• Участвовать в встречах и поездках
• Использовать все функции бота

До встречи в клубе! 🎉
```

*После отправки сообщения бот автоматически покидает группу через 5 секунд*

## 👤 Новый участник в группе

Когда в группу добавляют нового пользователя:

```
🎉 Добро пожаловать в клуб кабриолетов!

Привет, Иван! @ivan_petrov

🚗 Рады видеть нового любителя кабриолетов в нашем клубе!

Что делать дальше:
1️⃣ Напишите боту https://t.me/Cabrio_Ride_bot в личные сообщения
2️⃣ Используйте команду /start для начала работы
3️⃣ Пройдите регистрацию командой /register
4️⃣ Добавьте информацию о своих автомобилях

Возможности бота:
• Ведение профиля участника
• Каталог автомобилей клуба
• Система приглашений новых участников
• Поиск автомобилей по номерам

Если есть вопросы - обращайтесь к администраторам! 👋
```

## 🔧 Технические особенности

### Функции отправки
- `sendGroupNotification(message, options)` - текстовые уведомления
- `sendGroupPhoto(photoPath, caption, options)` - уведомления с фото

### Обработчики событий
- `bot.on('my_chat_member')` - добавление/удаление бота
- `bot.on('new_chat_members')` - новые участники группы

### Проверки безопасности
- Проверка ID группы перед отправкой
- Обработка ошибок при отправке
- Пропуск ботов при приветствии
- **Защита от чужих групп** - автоматическое покидание неправильных групп
- Информирование о правильной группе клуба

### Форматирование
- Markdown разметка для выделения
- Эмодзи для визуального разделения
- Структурированная подача информации

## 📊 Статистика уведомлений

Бот отправляет уведомления для:
- ✅ Добавления бота в группу (с защитой от чужих групп)
- ✅ Новых участников группы  
- ✅ Регистрации участников
- ✅ Первого автомобиля (полное приветствие)
- ✅ Дополнительных автомобилей
- ✅ Новых приглашений

## 🎯 Цели уведомлений

1. **Информирование** - держать всех в курсе активности клуба
2. **Мотивация** - поощрять активность участников
3. **Прозрачность** - показывать рост и развитие клуба
4. **Вовлечение** - привлекать внимание к новым участникам
5. **Отслеживание** - фиксировать важные события
