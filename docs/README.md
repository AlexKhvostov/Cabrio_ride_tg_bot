# 🚗 Cabrio Club Telegram Bot

Telegram бот для управления клубом кабриолетов с полной интеграцией с базой данных MySQL.

## 🚀 Возможности

- **Регистрация участников** клуба через Telegram
- **Управление профилем** участника
- **Добавление и редактирование** автомобилей
- **Система приглашений** новых участников
- **Статистика** клуба
- **Проверка членства** в чате перед предоставлением доступа
- **Загрузка фотографий** автомобилей и участников

## 📋 Требования

- Node.js 16+
- MySQL база данных
- Telegram бот (токен от @BotFather)
- ID чата для проверки участников

## ⚙️ Установка

1. **Клонируйте репозиторий и установите зависимости:**
```bash
npm install
```

2. **Создайте файл .env в корне проекта:**
```env
# Telegram Bot Configuration
BOT_TOKEN=your_bot_token_here
CHAT_ID=your_chat_id_here

# Database Configuration (MySQL)
DB_HOST=localhost
DB_USER=your_db_user
DB_PASSWORD=your_db_password
DB_NAME=your_db_name
DB_PORT=3306

# Optional: Admin IDs (comma separated)
ADMIN_IDS=123456789,987654321
```

3. **Убедитесь, что таблицы в MySQL созданы:**
   - Используйте файл `tables/create_tables.sql` для создания необходимых таблиц
   - Либо запустите существующий скрипт создания таблиц

## 🔧 Настройка

### Получение токена бота:
1. Напишите @BotFather в Telegram
2. Создайте нового бота командой `/newbot`
3. Скопируйте полученный токен в `.env`

### Получение ID чата:
1. Добавьте бота в группу клуба
2. Напишите любое сообщение в группе
3. Перейдите по ссылке: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
4. Найдите `chat.id` в ответе и скопируйте в `.env`

## 🚀 Запуск

```bash
# Запуск в продакшене
npm start

# Запуск в режиме разработки (с автоперезагрузкой)
npm run dev
```

## 📱 Использование

### Основные команды:
- `/start` - Запуск бота и проверка регистрации
- `/menu` - Главное меню с доступными функциями
- `/register` - Регистрация нового участника

### Функции через меню:
- **👤 Мой профиль** - Просмотр и редактирование профиля
- **🚗 Мои авто** - Список автомобилей участника
- **📝 Добавить авто** - Добавление нового автомобиля
- **📮 Мои приглашения** - История приглашений
- **🎯 Оставить приглашение** - Создание нового приглашения
- **📊 Статистика** - Статистика клуба
- **❓ Помощь** - Справка по использованию

## 🗄️ Структура базы данных

Бот использует три основные таблицы:

1. **members** - Участники клуба
2. **cars** - Автомобили участников
3. **invitations** - Приглашения новых участников

Подробное описание таблиц смотрите в папке `tables/`.

## 📁 Структура файлов

```
uploads/
├── members/          # Фотографии участников
│   └── {member_id}/
└── cars/            # Фотографии автомобилей
    └── {car_id}/
```

## 🔒 Безопасность

- Доступ к боту предоставляется только участникам указанного чата
- Все действия логируются в консоль
- Подключение к базе данных осуществляется через защищённые переменные окружения

## 🛠️ Разработка

Для добавления новых функций:

1. Добавьте новые callback_data в обработчик
2. Создайте соответствующие функции
3. При необходимости добавьте новые методы в `database.js`
