# 🛡️ Защита от добавления в чужие группы

## Описание функциональности

Бот клуба кабриолетов теперь имеет встроенную защиту от добавления в неправильные группы. Если бота добавляют не в ту группу, которая указана в конфигурации, он:

1. **Отправляет информационное сообщение** о том, что предназначен только для участников клуба кабриолетов
2. **Приглашает присоединиться** к правильной группе клуба
3. **Автоматически покидает** чужую группу через 5 секунд

## Как это работает

### При добавлении в правильную группу
```
🤖 Привет всем!

Я бот клуба кабриолетов! 🚗💨

Мои функции:
• 👤 Регистрация участников клуба
• 🚗 Управление автомобилями участников
• 🎯 Система приглашений новых участников
• 📊 Статистика и аналитика клуба
• 🔍 Поиск автомобилей по номерам

Как начать:
Напишите мне в личные сообщения команду /start

Важно: Я работаю только для участников этой группы!
🆔 ID группы: -1001613926754

Добро пожаловать в клуб! 🎉
```

### При добавлении в чужую группу
```
🤖 Привет!

Я бот клуба кабриолетов, но работаю только для участников нашей группы.

🚗 Если вы любитель кабриолетов, присоединяйтесь к нам:
👥 https://t.me/Cabrio_Ride

Там вы сможете:
• Общаться с единомышленниками
• Делиться фотографиями своих авто
• Участвовать в встречах и поездках
• Использовать все функции бота

До встречи в клубе! 🎉
```

После отправки этого сообщения бот автоматически покидает группу.

## Техническая реализация

### Обработчик события `my_chat_member`
```javascript
bot.on('my_chat_member', async (update) => {
    const { chat, new_chat_member } = update;
    
    if (new_chat_member.user.id === (await bot.getMe()).id && 
        new_chat_member.status === 'member') {
        
        if (chat.id.toString() === config.CHAT_ID.toString()) {
            // Наша группа - отправляем полное приветствие
            await bot.sendMessage(chat.id, welcomeMessage, { parse_mode: 'Markdown' });
        } else {
            // Чужая группа - отправляем информацию и покидаем
            await bot.sendMessage(chat.id, wrongGroupMessage, { parse_mode: 'Markdown' });
            
            setTimeout(async () => {
                await bot.leaveChat(chat.id);
            }, 5000);
        }
    }
});
```

### Улучшенная проверка доступа
Функция `checkAccess` также обновлена для информирования пользователей о правильной группе:

```javascript
async function checkAccess(msg) {
    const userId = msg.from.id;
    const isMember = await checkChatMembership(userId);
    
    if (!isMember) {
        bot.sendMessage(msg.chat.id, 
            '❌ Доступ запрещён!\n\n' +
            'Этот бот доступен только участникам клуба кабриолетов.\n\n' +
            '🚗 Присоединяйтесь к нашей группе:\n' +
            '👥 https://t.me/Cabrio_Ride\n\n' +
            'После вступления в группу все функции бота станут доступны!',
            { parse_mode: 'Markdown' }
        );
        return false;
    }
    
    return true;
}
```

## Конфигурация

Ссылка на группу клуба настраивается в `config.js`:

```javascript
CLUB: {
    name: 'Cabrio Club',
    description: 'Клуб любителей кабриолетов',
    welcomeMessage: '🚗 Добро пожаловать в Cabrio Club!',
    groupLink: 'https://t.me/Cabrio_Ride',
    botLink: 'https://t.me/Cabrio_Ride_bot'
}
```

## Логирование

Система ведет подробные логи для отладки:

```
🔍 Отладка my_chat_member:
Chat ID из события: -1001234567890
Chat ID из config: -1001613926754
Название группы: Тестовая группа
✅ Сообщение о неправильной группе отправлено
🚪 Покинули чужую группу: Тестовая группа
```

## Преимущества

1. **Автоматическая защита** - бот сам определяет правильность группы
2. **Информативность** - пользователи получают четкие инструкции
3. **Привлечение участников** - приглашение в правильную группу
4. **Чистота** - автоматическое покидание чужих групп
5. **Отладка** - подробное логирование для диагностики

## Безопасность

- Проверка ID группы происходит строго по конфигурации
- Задержка в 5 секунд позволяет пользователям прочитать сообщение
- Логирование помогает отслеживать попытки добавления в чужие группы
- Функция `leaveChat` обрабатывает ошибки gracefully
